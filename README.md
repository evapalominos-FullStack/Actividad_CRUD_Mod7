Se desarrolla un CRUD de la tabla clientes (rut, nombre, edad) en Node.js usando pg con consultas parametrizadas, validaciones básicas y manejo de errores. 

Se busca practicar inserción, actualización, borrado y consultas, con respuestas JSON coherentes que indiquen resultado y detalles de la operación.
